## OAuth账号切换

更改`.codex/auth.json`

```json
{
{
  "OPENAI_API_KEY": null,
  "last_refresh": "2026-06-15232Z",
  "tokens": {
    "access_token": "",
    "account_id": "39e543ed-4ffc-4649-8bff-9b4f8",
    "id_token": "",
    "refresh_token": ""
  }
}
}
```
如果`.codex/config.toml`配置了model_provider，则移除以下配置

```toml
model_provider = "codex_local_access"

[model_providers.codex_local_access]
name = "API名称"
base_url = "https://api.xxx.com/v1"
wire_api = "responses"
requires_openai_auth = true
experimental_bearer_token = "key"
supports_websockets = false

```
## API账号切换

更改`.codex/auth.json`

```json
{
  "OPENAI_API_KEY": "key",
  "auth_mode": "apikey"
}
```
更改`.codex/config.toml`配置了model_provider

```toml
model_provider = "codex_local_access"

[model_providers.codex_local_access]
name = "API名称"
base_url = "https://api.xxx.com/v1"
wire_api = "responses"
requires_openai_auth = true
experimental_bearer_token = "key"
supports_websockets = false

```
